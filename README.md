# Controle de Níveis de Água

Atividade da Agenda 11 da disciplina Desenvolvimento de Sistemas I.

## Descrição

Este projeto apresenta uma simulação simples de monitoramento de um reservatório de água, executada no terminal.

O programa exibe diferentes mensagens de alerta conforme o nível de água do reservatório, utilizando cores para facilitar a visualização das situações.

Para isso, foi utilizada a biblioteca `colorama`, que permite colorir as mensagens exibidas no terminal.

## Objetivo

O objetivo da atividade é aplicar conceitos básicos de programação em Python, como listas, funções, importação e uso de bibliotecas, estruturas condicionais e exibição de mensagens no terminal.

## Níveis do reservatório

| Nível | Situação | Cor utilizada |
|---|---|---|
| Nível 1 | Muito baixo (crítico) | Vermelho |
| Nível 2 | Baixo | Amarelo |
| Nível 3 | Médio | Verde |
| Nível 4 | Alto | Ciano |
| Nível 5 | Muito alto (alerta) | Azul |
